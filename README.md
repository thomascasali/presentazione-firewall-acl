# Firewall & Access Control List - Corso Completo

Presentazione interattiva su Firewall e ACL per il corso di **Sistemi e Reti** - Classe 5°

## Contenuti

| Modulo | Titolo | Argomenti |
|--------|--------|-----------|
| 1 | Fondamenti Firewall | Definizione, posizionamento, DMZ, principi (Default Deny) |
| 2 | Tipologie Firewall | Packet Filter, Stateful, Application Level, NGFW |
| 3 | Firewall Linux | iptables, nftables, NAT, Masquerading |
| 4 | Windows Firewall | Profili di rete, PowerShell, netsh |
| 5 | Access Control List | ACL Standard/Extended, Wildcard Mask, Named ACL |
| 6 | Quiz & Riepilogo | Quiz 15 domande, simulatori interattivi |

## Caratteristiche

- **45+ slide** interattive
- **4 simulatori** (Packet Filter, ACL, Wildcard Mask)
- **Quiz finale** con 15 domande e spiegazioni
- Design moderno con tema scuro
- Navigazione da tastiera (frecce, spazio, ESC)
- Responsive per tablet e desktop

## Tecnologie

- HTML5 + CSS3
- React 18 (via CDN)
- Babel (transpiler JSX)
- Google Fonts (Inter, JetBrains Mono, Space Grotesk)

## Come usare

1. Apri `index.html` nel browser
2. Naviga tra i moduli cliccando sulle card
3. Usa le frecce ← → o i pulsanti per navigare tra le slide
4. Premi ESC per tornare all'indice
